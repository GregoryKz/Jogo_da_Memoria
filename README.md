Jogo da Memória - HTML, CSS e JavaScript
Descrição

Este projeto é um Jogo da Memória interativo desenvolvido com HTML, CSS e JavaScript puro. O jogo utiliza cards com ícones de frutas que são embaralhados no início de cada partida. O objetivo é encontrar os pares correspondentes em um tabuleiro 4x4.

Funcionalidades

Tabuleiro 4x4 com 16 cards (8 pares de ícones).

Embaralhamento aleatório dos cards a cada partida.

Animação de flip ao clicar nos cards.

Cards que combinam permanecem destacados com estilo de match.

Cards que não combinam são revertidos automaticamente após um pequeno delay.

Responsivo: o tabuleiro se ajusta em telas menores.

Tecnologias Utilizadas

HTML5 para estrutura do jogo.

CSS3 com Flexbox e Grid Layout para o estilo e layout do tabuleiro.

JavaScript para a lógica do jogo, controle de cliques, comparação de pares e animações.
